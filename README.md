# agentic-war

AI 相关资源与工作流集合

## 简介

本项目用于存放 AI 相关的资源和工作流，目前包含 ComfyUI 工作流、Prompt 资料，以及把个人微信接入本地 AI Agent 的接入指南（cc-connect）。

## 目录结构

```
agentic-war/
├── cc-connect/                 # 个人微信 × 本地 Agent 接入指南（cc-connect）
├── comfyui/                    # ComfyUI 工作流目录
└── prompt/                     # Prompt 相关目录
    └── jailbreaking/           # 越狱 / Prompt 注入测试用例
```
